ðŸ“Œ What are Control Statements?

Control statements in C determine the flow of execution in a program. They help in decision-making, looping, and jumping to different parts of the program.

ðŸ”¹ Types of Control Statements

ðŸ“Œ 1. Conditional Statements (Decision-Making)

Used to execute a block of code based on a condition.

ðŸ‘‰ if Statement

Executes a block of code only if the condition is true.

    // Code to execute if condition is true
}

ðŸ‘‰ if-else Statement

Executes one block if the condition is true, otherwise executes another block.

if (condition) {
    // Code if condition is true
} else {
    // Code if condition is false
}

ðŸ‘‰ else-if Ladder

Used when there are multiple conditions to check.

if (condition1) {
    // Code if condition1 is true
} else if (condition2) {
    // Code if condition2 is true
} else {
    // Code if no conditions are true
}

ðŸ‘‰ switch Statement

Used to execute one case from multiple options.

    case value1:
        // Code for case 1
        break;
    case value2:
        // Code for case 2
        break;
    default:
        // Code if no case matches
}

ðŸ”¹ 2. Looping Statements

Loops execute a block of code multiple times based on a condition.

ðŸ‘‰ for Loop

Used when the number of iterations is known.

    // Code to execute
}

ðŸ‘‰ while Loop

Executes as long as the condition is true.

while (condition) {
    // Code to execute
}

ðŸ‘‰ do-while Loop

Executes at least once, then continues based on condition.

do {
    // Code to execute
} while (condition);

ðŸ”¹ 3. Jump Statements

Jump statements transfer control to another part of the program.

ðŸ‘‰ break Statement

Exits the loop or switch statement.

        break;  // Stops when i == 5

ðŸ‘‰ continue Statement

Skips the current iteration and moves to the next.

        continue;  // Skips when i == 3
    }
    printf("%d ", i);

ðŸ”¹ Important Keywords

  • int - Defines an integer variable.
  • float - Defines a floating-point variable.
  • char - Defines a character variable.
  • if, else - Used for conditional statements.
  • for, while, do - Used for loops.
  • return - Returns a value from a function.
  • void - Specifies that a function returns no value.
  • printf(), scanf() - Used for input/output.

ðŸ”¹ Data Types in C

  • int - Stores integers (e.g., 1, 2, 3).
  • float - Stores decimal numbers (e.g., 3.14).
  • char - Stores a single character (e.g., 'A').
  • double - Stores larger decimal numbers.

ðŸ”¹ Control Statements

Control the flow of a program:

  • if (condition) - Executes code if the condition is true.
  • else - Executes code if the condition is false.
  • switch - Selects one case from multiple options.
  • for, while, do-while - Looping statements.

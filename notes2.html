Data types define the type and size of data stored in a variable.

Data Type Size Range Example

ðŸ“Œ Type Modifiers

Type modifiers change the properties of basic data types.

  • short - Reduces the size of an integer. Example: short int a;
  • long - Increases the size of an integer. Example: long int b;
  • signed - Allows negative and positive values.
  • unsigned - Allows only positive values.

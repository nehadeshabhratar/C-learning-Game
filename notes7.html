ðŸ”¹ What is an Array?

An array is a collection of elements of the same data type stored in contiguous memory locations. It allows storing multiple values using a single variable name.

ðŸ”¹ Declaring and Initializing an Array

ðŸ“Œ Declaration

dataType arrayName[size];

Example:

int numbers[5];  // Declares an integer array with 5 elements

ðŸ“Œ Initialization

int numbers[5] = {1, 2, 3, 4, 5};  // Assigning values during declaration
int numbers[] = {10, 20, 30, 40}; // Size can be omitted when values are given

ðŸ”¹ Accessing Array Elements

Array elements are accessed using an index, starting from **0**.

#include 
int main() {
    int arr[3] = {10, 20, 30};
    printf("First element: %d", arr[0]);  // Output: 10
    return 0;
}

ðŸ”¹ Types of Arrays

ðŸ“Œ 1. One-Dimensional Array

A linear list of elements:

#include 
int main() {
    int arr[4] = {1, 2, 3, 4};
    for (int i = 0; i < 4; i++) {
        printf("%d ", arr[i]);
    }
    return 0;
}

ðŸ“Œ 2. Multi-Dimensional Arrays

An array of arrays, commonly a **2D array (Matrix)**:

#include 
int main() {
    int matrix[2][3] = {{1, 2, 3}, {4, 5, 6}};
    printf("Element at [1][2]: %d", matrix[1][2]);  // Output: 6
    return 0;
}

ðŸ”¹ Array Operations

ðŸ“Œ 1. Traversing an Array

Using a loop to access all elements:

#include 
int main() {
    int arr[5] = {10, 20, 30, 40, 50};
    for (int i = 0; i < 5; i++) {
        printf("%d ", arr[i]);
    }
    return 0;
}

ðŸ“Œ 2. Array Input from User

#include 
int main() {
    int arr[3];
    printf("Enter 3 numbers: ");
    for (int i = 0; i < 3; i++) {
        scanf("%d", &arr[i]);
    }
    printf("You entered: ");
    for (int i = 0; i < 3; i++) {
        printf("%d ", arr[i]);
    }
    return 0;
}

ðŸ“Œ 3. Finding the Largest Element in an Array

#include 
int main() {
    int arr[5] = {10, 45, 22, 78, 99};
    int max = arr[0];
    for (int i = 1; i < 5; i++) {
        if (arr[i] > max) {
            max = arr[i];
        }
    }
    printf("Largest element: %d", max);
    return 0;
}

ðŸ”¹ Strings as Character Arrays

In C, strings are arrays of characters ending with a null character ('\0').

    return 0;
